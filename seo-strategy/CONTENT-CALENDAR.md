# Content Calendar

## Cadence

2-3 new pieces/week during Phase 1-2 (weeks 1-12), tapering to 1-2/week in Phase 3-4 as comparison/localization work absorbs capacity. Sustainable for a small team without sacrificing the E-E-A-T bar (real screenshots, real testing, not spun content).

## Priority 1 — Comparison/Alternative Pages (Weeks 1-4)

Highest intent, direct competitive capture. Build as landing pages, not blog posts (see `SITE-STRUCTURE.md`).

1. `/smallpdf-alternative` — lead with the 2-tasks/day free-tier limit
2. `/ilovepdf-alternative` — lead with the 25MB file cap + paywall
3. Expand `/compare` into a hub with a real comparison table (iCreatePDF vs. iLovePDF vs. Smallpdf vs. PDF24: price, file limits, upload model, offline option)
4. `/pdf24-alternative` — lead with browser-vs-desktop-install distinction (lower priority, smaller gap)

## Priority 2 — Blog Gap-Fill for Unpaired Tools (Weeks 3-10)

23 of 51 tools currently have no dedicated how-to post: `organize-pdf`, `rotate-pdf`, `pdf-to-jpg`, `scan-to-pdf`, `pdf-to-text`, `edit-pdf`, `verify-signature`, `html-to-pdf`, `markdown-to-pdf`, `compare-pdf`, `crop-pdf`, `repair-pdf`, `header-footer`, `resize-pdf`, `qr-to-pdf`, `barcode-to-pdf`, `delete-pdf-pages`, `txt-to-pdf`, `ris-to-pdf`, `read-aloud-pdf`, `certify-pdf`, `pdf-to-excel`, `pdf-accessibility-checker`.

Sequence by estimated search volume (highest first — verify with `seo-dataforseo` keyword volume before finalizing order):
1. `pdf-to-jpg`, `rotate-pdf`, `pdf-to-text`, `edit-pdf` (highest-volume generic actions)
2. `html-to-pdf`, `pdf-to-excel`, `crop-pdf`, `resize-pdf`, `repair-pdf`
3. `qr-to-pdf`, `barcode-to-pdf`, `delete-pdf-pages`, `scan-to-pdf`, `organize-pdf`
4. Remaining niche tools (`verify-signature`, `certify-pdf`, `markdown-to-pdf`, `txt-to-pdf`, `ris-to-pdf`, `read-aloud-pdf`, `pdf-accessibility-checker`, `header-footer`, `compare-pdf`) — lower volume, but each closes a sitemap gap and supports topical completeness.

Each post: real screenshots of the actual tool, step-by-step instructions, FAQ block (targets AI Overviews/GEO), embedded link to the live tool.

## Priority 3 — Privacy/Trust Content (Weeks 5-8)
